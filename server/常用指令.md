# Flight-Monitor 常用指令速查

> `<IP>` = 你的服务器 IP（替换为实际地址）

---

## Windows 本机（Git Bash）

| 操作 | 指令 |
|------|------|
| SSH 登录服务器 | `ssh root@<IP>` |
| 上传数据库 | `scp flight_monitor.db root@<IP>:/opt/flight-monitor-server/` |
| 上传整个 server 文件夹 | `scp -r server root@<IP>:/opt/flight-monitor-server/` |
| 上传单个文件 | `scp server/static/index.html root@<IP>:/opt/flight-monitor-server/static/` |
| 生成密钥（免密登录） | `ssh-keygen -t ed25519 -C "flight-monitor" -f ~/.ssh/id_ed25519 -N ""` |
| 上传公钥 | `ssh-copy-id root@<IP>` |

## 服务器（SSH 登录后）

| 操作 | 指令 |
|------|------|
| 看板服务状态 | `systemctl status flight-monitor-server --no-pager` |
| 重启看板 | `systemctl restart flight-monitor-server` |
| 看日志 | `journalctl -u flight-monitor-server -f` |
| 生成密码哈希 | `python3.8 -c "import hashlib; print(hashlib.sha256(input('密码: ').encode()).hexdigest())"` |
| 编辑用户 | `vi /opt/flight-monitor-server/users.json` |
| 查看端口占用 | `ss -tlnp | grep 80` |
| 测试 API | `curl -s http://127.0.0.1/api/routes -H "Authorization: Bearer <TOKEN>"` |
